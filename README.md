# 🐦 Flappy Bird 

A modern Flappy Bird clone developed using **HTML5 Canvas, CSS, and JavaScript**.

This project recreates the classic Flappy Bird experience with smooth gameplay, responsive controls, collectible coins, pause functionality, and persistent high-score tracking.

---

## 🎮 Features

- 🐦 Smooth bird physics (gravity & jumping)
- 🟢 Infinite procedurally generated pipes
- 💥 Collision detection
- 🪙 Coin collection system
- 🏆 Score and High Score tracking
- 💾 High Score saved using Local Storage
- ⏸ Pause and Resume (Press **P**)
- 🔄 Restart Game (Press **R**)
- 🔊 Jump sound effect
- 🎨 Animated bird and improved pipe graphics
- 🖥 Desktop-friendly responsive gameplay

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas API
- Local Storage

---

## 🎮 Controls

| Key | Action |
|------|--------|
| Space | Jump |
| Left Mouse Click | Jump |
| P | Pause / Resume |
| R | Restart (after Game Over) |

## 🚀 How to Run

1. Clone this repository

```
git clone https://github.com/YOUR_USERNAME/flappy-bird-deluxe.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

Or use the **Live Server** extension in VS Code.
